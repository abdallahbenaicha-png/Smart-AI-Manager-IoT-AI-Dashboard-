# 🧠 Smart AI Manager

An IoT + AI system that monitors environmental data and makes intelligent decisions in real-time.

## 🚀 Features
- Real-time temperature monitoring
- AI-based decision system
- Automatic actuator control
- Cloud dashboard (Firebase)
- ESP32 integration

## ⚙️ Tech Stack
- ESP32
- Firebase Realtime Database
- Python AI Engine
- HTML/CSS/JS Dashboard

## 🧠 AI Logic
If temperature > 30°C → Fan ON  
Else → Fan OFF
